# MassSpringModel

## Description
*The CA course* requires an implementation of Mass Spring Model using only openGL.  
*The purpose* of the course is practicing the ***implicit numeric simulation methods***.

## Depends
1. GLFW *with* GLAD
2. GLM
3.  dear imgui, v1.70 WIP
